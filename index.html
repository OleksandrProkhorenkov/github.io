Project Description

This is a multifunctional program for Windows that provides many features, such as: deleting files, managing autorun, cleaning up RAM and applying tweaks for Windows 10.

This program is as up-to-date as possible.The problems of lack of space due to storage of unnecessary files, incomplete configuration of the Windows system, inconvenient autorun management and lack of RAM are almost always encountered when using a computer for a long time. This program will be useful for anyone who uses a computer.

There are many programs with similar functionality, but they are aimed at solving a single problem. My program combines all the solutions, which makes it possible to do without using one program, and such programs are not always as effective.
